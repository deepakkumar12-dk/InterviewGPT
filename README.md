## 🚀 Run Locally

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

Create a `.env` file inside the backend folder:

```env
GEMINI_API_KEY=your_key
GROQ_API_KEY=your_key
```
